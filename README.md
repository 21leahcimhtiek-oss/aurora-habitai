# HabitForge AI

> Build life-changing habits with AI coaching

## Features
- AI habit coaching
- Streak tracking
- Daily check-ins
- Progress analytics
- Smart reminders

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
